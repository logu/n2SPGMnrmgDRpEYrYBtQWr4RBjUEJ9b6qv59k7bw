## Usage

Just clone this repository, install dependencies and start application:

### Clone and install dependencies
```bash
git clone https://github.com/logu/n2SPGMnrmgDRpEYrYBtQWr4RBjUEJ9b6qv59k7bw logu-test
cd logu-test
npm install
```
### Start MongoDB
```bash
mongod
```

### Run project
```bash
npm run start
```
